# Universidata — Style Guide & Assets

[![Logo Horizontal](LOGO_HORIZONTAL/1x/universidata_logo_color_white_1x.png)](LOGO_HORIZONTAL/)

---

Universidata — Style Guide & Assets is the central repository for all official brand assets. It covers two brands: **UniversiData** (the company — full logo system across three formats: symbol, horizontal, vertical, in color and mono, for light and dark backgrounds) and **empowerEd** (the product suite — wordmark, the modular product lockups *School Manager* and *Course Studio*, and the symbol, each in black, white and gold). Every logo ships in multiple PNG resolutions and SVG, on transparent backgrounds. It also provides shared document templates (letterhead, invoice) you can copy and work on.

Use the interactive brand kit to preview every logo variant, download the right file for your use case, and grab a fresh copy of a document template.

**→ [Open Brand Kit](https://universidata-srl.github.io/style-guide-and-assets/)**

The kit is organised into tabs: **UniversiData** (logos), **empowerEd** (product-suite logos), **Templates** (Google Docs), **Wallpapers** (branded desktop backgrounds), **Fonts** (the brand typefaces), and **Brand Strategy**. Template buttons open a "Make a copy" dialog so each person gets their own editable copy — the master is never touched. Wallpaper previews are lightweight; the download buttons serve the full-resolution PNGs from the repo. The Fonts tab shows live specimens and offers each family as a `.zip` download.

The brand typefaces — **Barlow** (sans-serif) and **Playfair Display** (serif), both open-source under the OFL — are also applied to the kit page itself.

## Naming Convention

```
universidata_{type}_{color}_{variant}_{size}.png

color   →  color / mono
variant →  black (light backgrounds) / white (dark backgrounds)
size    →  0.25x · 0.5x · 1x · 2x · 4x · SVG
```
